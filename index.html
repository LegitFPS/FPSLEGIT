MISSING OBB

  • Step 1: Download the APK and OBB files from the links mentioned above.
  • Step 2: Enable the 'Install from Unknown Sources' option in your device.
    You can enable it by following these steps: Settings > Safety and Privacy >
    Allow Installation from Unknown Sources.
  • Step 3: Install the APK file, but don't open it yet.
  • Step 4: Copy the OBB files to - Android/OBB/com.tencent.ig
    (create a folder with this name if there isn't one already).
  • Step 5: You can launch the game.

If you come across any error, you can consider re-downloading the files and following
the steps mentioned above. ( FORCE CLOSE GAME )
